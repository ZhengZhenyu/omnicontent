## 概述

数据分析模块提供内容统计、发布统计和趋势分析功能。

---

## 端点列表

### 统计数据

- `GET /api/analytics/contents` - 内容统计
- `GET /api/analytics/channels` - 渠道统计
- `GET /api/analytics/dashboard` - 仪表板数据
